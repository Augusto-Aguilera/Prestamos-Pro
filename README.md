# 💰 Prestamos Pro | PATRIC SOFT ®

Sistema profesional de gestión de créditos personales, cobros recurrentes y control de empeños, diseñado específicamente para optimizar la rentabilidad y el orden financiero.

> "Transformá tu cuaderno de anotaciones en una financiera digital de alto rendimiento."

## 🎯 El Problema que Resolvemos
Muchos prestamistas y casas de empeño pierden dinero por falta de seguimiento, olvidos en los vencimientos o errores en el cálculo de saldos. **Prestamos Pro** elimina la fricción y centraliza toda la cartera de clientes en el celular del administrador.

## 🚀 Funcionalidades de Élite

* **💳 Control de Saldos Dinámico**: El sistema permite registrar abonos parciales y descuenta automáticamente del saldo pendiente en tiempo real.
* **📅 Gestión de Frecuencias**: Soporta cobros Diarios, Semanales, Quincenales y Mensuales.
* **🚨 Semáforo de Vencimientos**: Indicadores visuales automáticos:
    * 🟢 **Verde**: Al día o Crédito Cancelado.
    * 🟡 **Amarillo**: Vence hoy.
    * 🔴 **Rojo**: Pago vencido (Mora).
* **🛡️ Control de Empeños**: Registro detallado de objetos dejados en garantía.
* **📲 Integración con WhatsApp**: Envío de recordatorios personalizados con un solo toque, incluyendo el saldo actualizado y la fecha de vencimiento.
* **💾 Persistencia Local**: Los datos se guardan en el dispositivo (LocalStorage), permitiendo acceso rápido sin necesidad de servidores externos complejos.

## 🛠️ Stack Tecnológico

* **Frontend**: HTML5, CSS3 (Diseño Premium Black & Gold).
* **Lógica**: JavaScript ES6+ (Manipulación de Arrays y Almacenamiento Local).
* **Iconografía**: FontAwesome 6.0 para una interfaz intuitiva.
